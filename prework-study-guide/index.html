HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is used to structure a webpage and its content.
  • Storage of big files are allowed because of the application cache feature.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The content area contains the real content of the element, such as text, an image, or a video player.
  • Border area extendds the padding area to include the element's borders
